# 🖋️ Tictactrip - Text Justify API

Projet réalisé dans le cadre du test technique Fullstack pour **Tictactrip**. Cette API en Node.js/TypeScript permet de justifier un texte brut à 80 caractères par ligne, tout en gérant une authentification par token et une limite de consommation quotidienne.

## 🚀 Fonctionnalités

- **Justification de texte** : Un algorithme personnalisé qui aligne les bords gauche et droit à exactement 80 colonnes.
- **Authentification** : Système de génération de token via email.
- **Rate Limiting** : Limite stricte de 80 000 mots par jour et par token pour éviter les abus.
- **TypeScript** : Code entièrement typé pour une meilleure maintenabilité.
- **ES Modules** : Utilisation des dernières normes de modules Node.js.
